Conseil

{% renderTemplate "md" %} ### Technologies sémantiques - Choix d'outils de traitement ou de triplestore - Publication de données en Linked Data - Modélisation d'ontologie, (re-)structuration de thesaurus ### Architectures pour rendre les contenus intelligents - Utilisation de vocabulaires contrôlés pour l'annotation des contenus - Annotation automatique ou semi-automatiques de contenus - Enrichissement de moteurs de recherche avec du vocabulaire métier ### Open Data - Choix de données et de vocabulaires à réutiliser - Accompagnement dans la publication de données "5 étoiles" - Accompagnement dans la réutilisation de données ouvertes {% endrenderTemplate %}

Développement

{% renderTemplate "md" %} ### Data-Ware - Reprise et migration de données vers les formats sémantiques - Modélisation et choix de vocabulaires - Réutilisation et exploitation de données ouvertes - Traitement et indexation des données dans un moteur de recherche ### Soft-ware - Intégration d'outils dans les architectures de traitement du contenu : text-mining, moteurs de recherche, triplestores, systèmes de gestion de thesaurus - Maintenance et évolutions sur les applications à base de technologies sémantiques - Architectures Java/J2EE {% endrenderTemplate %}
